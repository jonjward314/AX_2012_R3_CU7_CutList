Create layer changes

LOGIC CHANGE DEV writes Per_Series into cutlist rows from BOM quantity series.

Before: Cutlist row had Qty/Pieces, but no explicit Per_Series assignment.
After: Cutlist row includes Per_Series = BOMQtySerie.

This creates field-level data differences even when record counts are unchanged.

ReportDP changes

  • Cover/Summary now add explicit aggregate selection fields such as SUM(Pieces), SUM(Qty), MAX(PWSBuildScheduleId).
  • Detail remains row-granular, but DEV injects compacted metadata strings (build schedules and unit counts).
  • DEV includes additional temp output fields: Per_Series, PWSBuildScheduleId, UnitCounts.
