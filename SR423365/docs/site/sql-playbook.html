How to use

  • Start with the same filter scope stakeholders used in AX (schedule, prefix, model, etc.).
  • Paste the WHERE filter into /* your filter */.
  • Run Cover, Detail, and Summary SQL to compare shapes.

A) Cover (grouped/compacted)

WITH base AS (
  SELECT * FROM McaCutlist WHERE /* your filter */ 1=1
)
SELECT
  b.ModelItemId,
  b.ModuleItemId,
  b.Prefix,
  b.InventSiteId,
  b.ShipToWarehouse,
  b.DueDateTime,
  SUM(b.Qty) AS Qty,
  SUM(b.Pieces) AS Pieces,
  MAX(b.PWSBuildScheduleId) AS PWSBuildScheduleId,
  MAX(b.Per_Series) AS Per_Series
FROM base b
GROUP BY
  b.ModelItemId, b.ModuleItemId, b.Prefix,
  b.InventSiteId, b.ShipToWarehouse, b.DueDateTime
ORDER BY b.DueDateTime, b.Prefix, b.ModelItemId;

B) Detail (row-level + compacted metadata)

WITH base AS (
  SELECT * FROM McaCutlist WHERE /* your filter */ 1=1
), allSchedules AS (
  SELECT STRING_AGG(DISTINCT PWSBuildScheduleId, ' ') AS BuildSchedules
  FROM base
  WHERE PWSBuildScheduleId IS NOT NULL AND PWSBuildScheduleId <> ''
), unitCounts AS (
  SELECT ModelItemId, Prefix, COUNT(DISTINCT SalesLineRefRecId) AS UnitCount
  FROM base
  GROUP BY ModelItemId, Prefix
)
SELECT
  b.*,
  CAST(ROUND(b.Pieces * b.KitQty,0) AS int) AS DetailPieces,
  s.BuildSchedules,
  uc.UnitCount
FROM base b
CROSS JOIN allSchedules s
LEFT JOIN unitCounts uc ON uc.ModelItemId = b.ModelItemId AND uc.Prefix = b.Prefix
ORDER BY b.DueDateTime, b.Prefix, b.ModelItemId;

C) Summary (grouped/compacted)

WITH base AS (
  SELECT * FROM McaCutlist WHERE /* your filter */ 1=1
)
SELECT
  b.ModelItemId,
  b.ModuleItemId,
  b.CutGroupId,
  b.Prefix,
  b.ShipToWarehouse,
  b.DueDateTime,
  SUM(b.Qty) AS Qty,
  SUM(b.Pieces) AS Pieces,
  MAX(b.PWSBuildScheduleId) AS PWSBuildScheduleId,
  MAX(b.Per_Series) AS Per_Series
FROM base b
GROUP BY
  b.ModelItemId, b.ModuleItemId, b.CutGroupId,
  b.Prefix, b.ShipToWarehouse, b.DueDateTime
ORDER BY b.DueDateTime, b.Prefix, b.ModelItemId;

Equation translation for non-technical audiences

Summary/Cover (DEV): Rounded Total = Ceiling(Sum(KitQty))
Detail (DEV): Final Pieces = Ceiling( Sum(Pieces × KitQty) ÷ Per_Series )
If Per_Series is missing: divide by 1 and mark as “Not Set”.
These queries mirror report shaping intent, not every AX helper-side effect.
