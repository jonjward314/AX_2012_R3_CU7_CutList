Glossary

  • Cutlist: Manufacturing-oriented line-level instruction set for cutting materials.
  • DP/RDP: AX SSRS data provider class that prepares report dataset data.
  • Contract: SysOperation data contract carrying parameters and packed queries.
  • TempDB table: Temporary SQL-backed table used by report preprocessing.
  • Packed query: Base64-encoded serialized AX Query object.
  • UsageType: Cut material type enum (e.g., Linear, Sheet) used in detail grouping.
  • KitQty: Quantity multiplier from exploded model/module context.
  • Pieces: Piece count metric whose semantics can be raw or expanded by pipeline stage.
